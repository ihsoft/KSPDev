Remarks

Use this player when the source of the sound is a GUI object (e.g. a GUI control). This class implements all the boilerplate to load and play the sound resources. All the sounds are cached wihtin the scene, so repeating requests to the same sound won't add extra latency.

In case of the very first usage of the sound is latency restricted, the sound resource can be pre-cached via the CacheSound(String) method. It will increase the loading time, though.

Examples
Copy
public class UISoundPlayerDemo1 : PartModule {
  public override void OnAwake() {
    // To not loose the latency on the sound play pre-cache it.
    UISoundPlayer.instance.CacheSound("ooo.ogg");
  }

  public override void OnUpdate() {
    if (Input.GetKeyDown("O")) {
       UISoundPlayer.instance.Play("ooo.ogg");  // Played from cache. No delay.
    }
    if (Input.GetKeyDown("P")) {
      UISoundPlayer.instance.Play("ppp.ogg");  // May delay the game while loading the resource.
    }
  }
}
