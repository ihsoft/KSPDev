KSPDev.ConfigUtils Namespace

A set of utility methods and classes to make accessing config files easier. You may choose to work with each field (as you usually do in KSP) but with use of convenience methods. Or you may make use of meta programming approach and only add attributes to the fields that require persistence.

If you like dealing with settings on yourself then you may find useful ConfigAccessor helper class that simplifies routine operations with configuration nodes and files.

If you'd like to start using new attributes system that allows annotation of the settings fields you may start from PersistentFieldAttribute. This topic has good code examples that will help you to start.
